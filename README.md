# 🚀 Modern Personal Portfolio

Welcome to my personal portfolio website! This project is a showcase of my web development journey, featuring a clean, responsive, and interactive design to highlight my skills and projects.

## ✨ Features
* **Hero Section:** A visually appealing introduction with a professional profile photo.
* **Floating Skill Pills:** Dynamic icons for HTML5, CSS3, JavaScript, and GitHub with smooth floating animations.
* **Modern UI:** Built with a dark theme, glassmorphism effects, and vibrant gradients.
* **Responsive Design:** Fully optimized for different screen sizes.
* **Interactive Elements:** Hover effects, smooth transitions, and animated components.

## 🛠️ Technologies Used
I have utilized a modern tech stack to build this portfolio:
* **HTML5:** For structured and semantic web content.
* **CSS3:** For advanced styling, including Flexbox, CSS Variables, and Keyframe Animations.
* **Font Awesome:** For professional scalable vector icons.
* **Google Fonts:** Using 'Playfair Display' and 'Fira Code' for a premium typography feel.

## 🚀 Getting Started
To run this project locally, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Anees804804/my-complete-portfolio/new/main](https://github.com/Anees804804/my-complete-portfolio/new/main)
    ```
2.  **Navigate to the folder:**
    ```bash
    cd portfolio_check
    ```
3.  **Open the project:**
    Simply open `index.html` in your favorite web browser.
